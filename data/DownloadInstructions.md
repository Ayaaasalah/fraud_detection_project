#  Dataset Download Instructions

The raw dataset files for this project are too large to be hosted directly on GitHub. Please follow these steps to download the necessary data from the shared Google Drive folder and place them in the correct directory.

### 1. Access the Google Drive Folder

Click the link below to access the folder containing the four raw CSV data files:

** DATASET FOLDER LINK:** [https://drive.google.com/drive/folders/1QQm7x_cpbRGVqSEVe6a6y3TGrINwNXVF?usp=sharing](https://drive.google.com/drive/folders/1QQm7x_cpbRGVqSEVe6a6y3TGrINwNXVF?usp=sharing)

### 2. Download the Files

You must download all four of the following files:

* `Train-1542865627584.csv` (Provider labels)
* `Train_Beneficiarydata-1542865627584.csv` (Beneficiary data)
* `Train_Inpatientdata-1542865627584.csv` (Inpatient claims)
* `Train_Outpatientdata-1542865627584.csv` (Outpatient claims)
* 'integrated_provider_data_modeling.csv' (Feature engineered file)

### 3. File Organization

After downloading and extracting the files, ensure that they are placed inside a folder named **`data/`** located at the root level of your local repository.
